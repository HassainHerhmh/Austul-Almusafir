# تطبيق تتبع الباص (سائق)

تطبيق Android يرسل موقع GPS للمنصة أثناء الرحلة.

## المتطلبات
1. على السيرفر: `npx prisma db push` لإضافة `TripLocation` و`User.driverId`.
2. من الأدمن → المستخدمون: أنشئ حساباً بدور **سائق (تتبع)** واربطه بسجل سائق.
3. اربط السائق برحلة مفتوحة.

## التشغيل للتطوير
```bash
cd mobile
npm install
npx expo start
```

## بناء APK
```bash
cd mobile
npx eas-cli login   # إن لزم
npx eas build -p android --profile preview
```

أو محلياً بعد `npx expo prebuild`:
```bash
npx expo run:android --variant release
```

ملف [`eas.json`](eas.json) جاهز لملف APK قابل للتوزيع الداخلي.

## الاستخدام
1. السائق يدخل بحسابه.
2. يختار الرحلة → **بدء التتبع** (يظهر إشعار مستمر).
3. الأدمن يرى الموقع في **تتبع الباصات**.
4. عند الوصول: **إيقاف التتبع**.

> ملاحظة: إيقاف التطبيق قسرياً أو تقييد البطارية الشديد قد يوقف التتبع — أبقِ الإشعار ظاهراً.
